# Calculadora en C++ (Consola)

Este proyecto es una calculadora básica en C++ usando arquitectura en capas y programación orientada a objetos.

## Funcionalidades
- Suma
- Resta
- Multiplicación
- División (con validación de división por cero)

## Estructura
- `main.cpp`: Interfaz de consola
- `Calculadora.h / .cpp`: Lógica de operaciones
